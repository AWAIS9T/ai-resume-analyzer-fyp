# AI-Based Resume Analyzer & Job Recommendation System

## Project Overview
This project is a Final Year Project (FYP) based on a web application using MERN Stack and Python.

The system analyzes resumes using Artificial Intelligence techniques and recommends suitable jobs based on extracted skills.

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Python (AI / NLP)

## Objectives
- Analyze resumes automatically
- Extract skills using AI
- Recommend relevant jobs
- Provide resume scoring

## Future Scope
- Improve AI accuracy
- Add more job categories
- Deploy system online
